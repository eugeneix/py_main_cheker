# Как получить правильный Chat ID для группы/канала

## Проблема: "Chat not found"

Если вы получаете ошибку "Chat not found", это означает, что:
1. Бот не добавлен в группу/канал
2. Бот не является администратором (для каналов обязательно)
3. Chat ID неправильный

## Решение: Получить Chat ID правильно

### Способ 1: Через бота @RawDataBot (самый надежный)

1. **Добавьте бота @RawDataBot в вашу группу/канал**
   - Найдите [@RawDataBot](https://t.me/RawDataBot) в Telegram
   - Добавьте его в группу/канал

2. **Отправьте любое сообщение в группу/канал**

3. **Бот покажет JSON с данными**
   - Найдите в ответе строку с `"chat":`
   - Там будет `"id":-1001234567890` (отрицательное число)
   - **Скопируйте это число полностью, включая минус**

### Способ 2: Через API Telegram

1. **Убедитесь, что бот добавлен в группу/канал**

2. **Отправьте любое сообщение в группу/канал** (от вас или от бота)

3. **Откройте в браузере:**
   ```
   https://api.telegram.org/bot8208055846:AAFNFYKN4mPfvO4aPQ13xOr4aqGw_ayuyCA/getUpdates
   ```

4. **Найдите в ответе:**
   ```json
   "chat":{
     "id":-1001234567890,
     "title":"Название группы",
     "type":"supergroup"
   }
   ```

5. **Скопируйте число после `"id":`** (включая минус, если есть)

### Способ 3: Через бота @userinfobot

1. **Добавьте [@userinfobot](https://t.me/userinfobot) в группу/канал**

2. **Отправьте `/start` в группе/канале**

3. **Бот вернет Chat ID** (отрицательное число)

## Важно: Проверьте настройки бота

### Для группы:
1. ✅ Бот должен быть добавлен в группу
2. ✅ Бот должен иметь права на отправку сообщений
3. ✅ (Опционально) Сделайте бота администратором

### Для канала:
1. ✅ Бот должен быть добавлен в канал
2. ✅ **Бот ОБЯЗАТЕЛЬНО должен быть администратором канала**
3. ✅ Бот должен иметь права на отправку сообщений

## Как добавить бота как администратора канала:

1. Откройте настройки канала
2. Перейдите в "Администраторы"
3. Нажмите "Добавить администратора"
4. Найдите вашего бота (`@main_page_monitor_bot`)
5. Добавьте его и дайте права на отправку сообщений

## Проверка Chat ID

После получения Chat ID, проверьте его:

1. **Chat ID должен быть отрицательным числом** для групп/каналов
   - Пример: `-1001234567890`
   - НЕ: `3294239202` (это личный чат)

2. **Для супергрупп и каналов** Chat ID начинается с `-100`
   - Пример: `-1001234567890`

3. **Для обычных групп** Chat ID начинается с `-`
   - Пример: `-123456789`

## Тест отправки сообщения

После получения Chat ID, протестируйте отправку:

```python
import asyncio
from telegram import Bot

async def test():
    bot = Bot(token="8208055846:AAFNFYKN4mPfvO4aPQ13xOr4aqGw_ayuyCA")
    chat_id = "-3294239202"  # Ваш Chat ID
    await bot.send_message(chat_id=chat_id, text="Тестовое сообщение")
    print("Сообщение отправлено!")

asyncio.run(test())
```

Если сообщение отправлено успешно - Chat ID правильный!
