# Отправка уведомлений в группу или канал Telegram

## Способ 1: Отправка в группу

### Шаг 1: Создайте группу или используйте существующую
1. Откройте Telegram
2. Создайте новую группу или выберите существующую
3. Добавьте вашего бота в группу (найдите бота по username и добавьте)

### Шаг 2: Сделайте бота администратором (опционально, но рекомендуется)
1. Откройте настройки группы
2. Перейдите в "Администраторы"
3. Добавьте вашего бота как администратора

### Шаг 3: Получите Chat ID группы

**Способ A: Через бота @userinfobot**
1. Добавьте [@userinfobot](https://t.me/userinfobot) в группу
2. Отправьте в группе команду `/start`
3. Бот вернет Chat ID группы (отрицательное число, например: `-1001234567890`)

**Способ B: Через API**
1. Отправьте любое сообщение в группу (от вашего бота или от вас)
2. Откройте в браузере (замените `ВАШ_ТОКЕН` на токен бота):
   ```
   https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
   ```
3. Найдите в ответе `"chat":{"id":-1001234567890}` (отрицательное число)
4. **Скопируйте это число** - это Chat ID группы

**Способ C: Через бота @RawDataBot**
1. Добавьте [@RawDataBot](https://t.me/RawDataBot) в группу
2. Отправьте любое сообщение в группе
3. Бот покажет JSON с данными, найдите `"id":-1001234567890`

### Шаг 4: Используйте Chat ID группы при запуске

```bash
python web_monitor.py \
  "https://www.rtoperator.ru/" \
  "ВАШ_ТОКЕН" \
  "-1001234567890" \
  "auto" \
  "Рождество 2026"
```

**Важно:** Chat ID группы - это отрицательное число!

## Способ 2: Отправка в канал

### Шаг 1: Создайте канал или используйте существующий
1. Откройте Telegram
2. Создайте новый канал или выберите существующий
3. Добавьте вашего бота в канал как администратора

### Шаг 2: Добавьте бота как администратора канала
1. Откройте настройки канала
2. Перейдите в "Администраторы"
3. Нажмите "Добавить администратора"
4. Найдите вашего бота и добавьте его
5. Дайте боту права на отправку сообщений

### Шаг 3: Получите Chat ID канала

**Способ A: Через форвард сообщения**
1. Отправьте любое сообщение в канал
2. Перешлите это сообщение боту [@userinfobot](https://t.me/userinfobot)
3. Бот вернет Chat ID канала (отрицательное число, например: `-1001234567890`)

**Способ B: Через API**
1. Отправьте любое сообщение в канал
2. Откройте в браузере (замените `ВАШ_ТОКЕН` на токен бота):
   ```
   https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
   ```
3. Найдите в ответе `"chat":{"id":-1001234567890}` (отрицательное число)
4. **Скопируйте это число** - это Chat ID канала

**Способ C: Через публичный канал**
Если канал публичный, его Chat ID можно получить так:
- Username канала: `@channel_name`
- Chat ID будет: `-100` + ID канала

### Шаг 4: Используйте Chat ID канала при запуске

```bash
python web_monitor.py \
  "https://www.rtoperator.ru/" \
  "ВАШ_ТОКЕН" \
  "-1001234567890" \
  "auto" \
  "Рождество 2026"
```

**Важно:** Chat ID канала - это отрицательное число!

## Примеры Chat ID

- **Личный чат:** `123456789` (положительное число)
- **Группа:** `-1001234567890` (отрицательное число, начинается с `-100`)
- **Супергруппа:** `-1001234567890` (отрицательное число)
- **Канал:** `-1001234567890` (отрицательное число)

## Проверка работы

После настройки:

1. Запустите монитор
2. Отправьте тестовое сообщение в группу/канал от бота
3. Проверьте, что бот может отправлять сообщения

Если бот не может отправить сообщение:
- Убедитесь, что бот добавлен в группу/канал
- Проверьте, что бот является администратором (для каналов обязательно)
- Проверьте правильность Chat ID (должно быть отрицательное число для групп/каналов)
